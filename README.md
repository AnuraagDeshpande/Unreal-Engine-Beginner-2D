Beginner 2D Game made in Unreal Engine 5 using paper 2D to create a slide scrolling character and stage from a tile map, made using blueprints. All paper assets dervied from the popular legacy fantasy pack.

Future steps:
- Add more levels
- Add music
- Experiment with different assets
#

This project focuses on setting up responsive character physics, 2D sprite handling (Paper2D), and clean input mapping.
